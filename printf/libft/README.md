# LIBFT

La libreria creada en este proyecto es la libft, la cual contiene implmentaciones propias de funciones de string.h para poder interactuar con los string, otras funciones adicionales para manejar string y ficheros, las cuales no estan incluidas en string.h, y funciones para la creacion y manejo de listas enlazadas.
## Instrucciones
Para ejecutar el programa debes ejecutar el comando make sobre la carpeta y luego compilar la libreria esatica (libft.a) con un main para poder tener un ejecutable.
