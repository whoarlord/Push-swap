# Printf
## Descripcion
Este proyecto es una implementacion propia de de la funcion printf. En el caso de este proyecto solo se tomaran en cuenta los siguientes formatos: cspdiuxX% de la funcion original. en el siguiente link obtendran mas informacion acerca de cada uno de estos caracteres: 
- Printf format specifiers: https://cplusplus.com/reference/cstdio/printf/
## Instrucciones
Para poder hacer uso de la libreria de printf, deberas ejecutar el comando make sobre la raiz de la carpeta y compilar el archivo libftprintf.a obtenido junto a un main o el programa a utilizar.
